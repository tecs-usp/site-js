## Vamos começar 🛠️

### Configurando o projeto 🔧

Primeiramente, instale as dependências.

```bash
pnpm install
```

### Executando o projeto ▶️

Na raiz do projeto, execute

```bash
pnpm run dev
```

Então, acesse [http://localhost:3000](http://localhost:3000) no seu browser.
